Applying and Understanding CSS

CSS is a code language that can not be of much use on its own. It is designed to style and "beautify" your websites so they aren't so plain and bland with just you HTML code. CSS stands for "Cascading Style Sheets"; that means that your CSS code will style your HTML with declarations that identify and style elements in a orderly manner.

You write CSS by selecting an element by its name: for an <h1> tag you would name your declaration h1{} and fill the curly brakets with as many declarations as you want.


h1(my ruleset target){
my declaration: my value;
}


You can use the name of the element or you can give elements names and use those. The attribute of class can give lots of elements a name that can be styled together. The id attribute can give an element a name that can be styled separately from all other classes or element names it falls into.

For the class attribute, put a period (.) before the name when styling it in your CSS code.


.MyClass(my ruleset target){
my declaration: my value;
}


For the id attribute, put a hash-tag (#) at the begginning of your id name in your CSS code.


#MyId(my ruleset target){
my declaration: my value;
}
